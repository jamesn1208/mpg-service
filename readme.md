# Fuel Service
This service allows you to log your MPG for a given refuel & draw heuristic analysis based on previous data
nodes. It uses a `Vue.js` client, `FastAPI` server, `nginx` proxy and a `PostgreSQL` database - all orchestrated by Docker Compose.

## Deployment
Simply run `docker compose up --build -d`, and head to `http://~/app/` to use it!
